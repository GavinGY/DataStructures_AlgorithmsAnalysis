DSAA-Project  Data Structures Algorithms Analysis
个人开发和维护的一个C语言项目工程
主属应用层和算法层（因为采用C语言，也可以用于裸机，驱动，内核和系统层，人机层）
主要目的是解决各种嵌入式设备实际应用问题和算法问题（包括设备功能，设备生产等）（因为采用C语言，也可以用于解决人机信息终端设备，服务器设备问题）

1.集成多种算法和数据结构实现模块
2.集成多个项目实际应用Demo
3.不断通过维护拓展系统支持功能和优化系统性能
4.可以通过自动化配置工具(后期加入)进行快速裁剪和移植到目标平台
5.支持多个平台（Windows/Linux, MCU-Bare, 嵌入式RTOS，嵌入式Linux）

工程维护的编译平台：Ubuntu，GCC，Makefile

工程特点：功能模块化，系统框架化，文档化 
1. 在学习与工作中不断的丰富和维护这个project；
2. 同时可以将本project应用于解决实际问题（通过 tailor,porting,modify,add ） 人工裁剪，脚本裁剪；人工移植，脚本配置。

--------------------------------------------------------------------------------------

Big Module:

1.七大排序算法实现  接口：include/sorting_algorithm.h 选择排序和快速排序

2.C语言链表实现     接口：include/linked_list.h

3.数组逆序算法实现  接口：include/arry_reverse.h

4.文件操作（二进制和文本）  

5.选择问题的算法实现

6.查找问题的算法实现：二分法

7.Debug功能         接口：include/debug.h

--------------------------------------------------------------------------------------

Little Module:

1.函数执行时间计算

2.CPU位数判断和大小端判断

3.递归的函数实现

4.随机数生成函数
